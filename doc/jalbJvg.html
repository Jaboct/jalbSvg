Rendering

Both Jvg and Svg render from (render_svg.c).

only Jvg uses (jRender.svg).

Standard function looks like:
void ELEMENT_render ( int *screenDims, GLuint *glBuffers, int *XYWHpass, struct ELEMENT *ele, float *viewLoc, float viewScale );
With a few global varaibles:
int selected; says if this or an element beneith it is selected.
int thisSel; says if this specifically, and not something beneith it is selected.
